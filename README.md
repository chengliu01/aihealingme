## 设计特色

### 动态渐变背景
- 全局背景使用蓝紫粉青四色渐变
- 20秒循环动画，柔和流畅
- 营造宁静而富有生命力的氛围

### 玻璃态设计（Glassmorphism）
- 半透明毛玻璃卡片
- 配合动态背景创造层次感
- 精致的边框和高光

### 生动的动画效果
- **Hero 区域**：多层动态光晕 + 浮动表情
- **按钮交互**：缩放 + 渐变变化
- **卡片悬停**：上浮 + 阴影扩散
- **聊天气泡**：流畅的消息进入动画

### 渐变配色方案
- **主强调色**：紫罗兰 (#8b5cf6) → 天蓝 (#0ea5e9)
- **按钮渐变**：紫色到蓝色的优雅过渡
- **进度条**：动态渐变填充
- **文字渐变**：紫蓝双色渐变标题

## 核心功能

### 1. 单次疗愈
- 情绪状态多选（胶囊按钮）
- 场景选择
- 情绪强度滑块（带渐变填充）
- AI 分析和音频生成

### 2. 规划疗愈
- 聊天式多轮对话
- 阶段性疗愈计划预览
- 计划进度追踪

### 3. 社区功能
- 视频卡片流
- 分类筛选和搜索
- 评论互动

## 技术栈

- **React 18** + **TypeScript**
- **Tailwind CSS** - 自定义动画和渐变
- **Framer Motion** - 流畅交互动画
- **Zustand** - 状态管理

## 启动项目

```bash
npm install
npm run dev
```

访问 http://localhost:3000

## 设计亮点

1. **无需"AI 驱动的个性化疗愈"标签** - 界面更简洁
2. **整体渐变背景** - 生动而不刺眼
3. **悬浮表情装饰** - 增添情感温度
4. **玻璃态卡片** - 现代高级感
5. **渐变按钮和标签** - 视觉层次分明
